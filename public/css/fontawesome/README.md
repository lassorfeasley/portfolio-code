# Font Awesome Pro 6.7.2

## Setup Instructions

Copy the following files from your downloaded `fontawesome-pro-6.7.2-web` folder:

1. **CSS files**: Copy all CSS files from `fontawesome-pro-6.7.2-web/css/` to this directory (`site/public/css/fontawesome/`)
   - Specifically, you need `all.min.css` (or `all.css`)
   - You may also want `all.css` if you prefer the non-minified version

2. **Webfonts**: Copy all font files from `fontawesome-pro-6.7.2-web/webfonts/` to `site/public/webfonts/`
   - This includes files like:
     - `fa-solid-900.woff2`
     - `fa-solid-900.woff`
     - `fa-regular-400.woff2`
     - `fa-regular-400.woff`
     - And other font files as needed

## Quick Copy Commands

If your downloaded Font Awesome is in your Downloads folder, you can run:

```bash
# Copy CSS files
cp -r ~/Downloads/fontawesome-pro-6.7.2-web/css/* /Users/lassor/portfolio-code/site/public/css/fontawesome/

# Copy webfonts
cp -r ~/Downloads/fontawesome-pro-6.7.2-web/webfonts/* /Users/lassor/portfolio-code/site/public/webfonts/
```

After copying the files, your command icon and all other Font Awesome Pro icons should work!
